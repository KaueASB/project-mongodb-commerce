# Boas-vindas ao repositório do Projeto MongoDB Commerce! 

Este projeto tem cunho educacional. Foi desenvolvido para colocar em prática o ensino dado pela Trybe.

<details>
<summary><strong>👨‍💻 O que foi desenvolvido</strong></summary><br />

  Neste projeto o objetivo é realizar consultas simples e complexas, filtros, inclusão e exclusão de dados no **MongoDB**!

  Nesse projeto, trabalhamos com o banco de dados `commerce`, que contém dados do cardápio do **McDonald's**, como ingredientes, valores nutricionais e dados fictícios de vendas.
